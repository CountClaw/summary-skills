# 3-2-1策略

## 概述

3-2-1策略是一种简洁的反思总结框架，通过数字约束（3-2-1）引导总结者快速聚焦核心内容。它结构简单、易于上手，广泛用于课堂教学、阅读反思和日常学习中。

---

## 基本形式

| 数量 | 内容 | 目的 |
|------|------|------|
| **3** | 学到的3个要点 | 提取核心知识 |
| **2** | 觉得有趣的2件事 | 激发好奇心和连接 |
| **1** | 仍然存在的1个疑问 | 识别知识盲区 |

---

## 详细说明

### 3 — 学到的三个要点

**目的：** 提取最重要的学习成果

**操作：**
- 写下你认为最重要的3个知识点/观点/技能
- 每个要点用1-2句话表达
- 不追求全面，只写最核心的

**示例：**
```
3个学到的要点：
1. 习惯的养成依赖四个步骤：提示→渴望→反应→奖励
2. 环境设计比意志力更重要——改变环境比改变自己更容易
3. 习惯叠加法：将新习惯绑定在已有习惯之后（"做完X之后，我做Y"）
```

---

### 2 — 觉得有趣的两个发现

**目的：** 激发好奇心，建立情感连接

**操作：**
- 写下2个让你感到有趣、惊讶或印象深刻的点
- 可以是颠覆认知的发现、有趣的案例、意想不到的联系

**示例：**
```
2个有趣的发现：
1. 英国自行车队通过"1%的微小改进"（如洗手方式、枕头选择）在10年内
   从默默无闻变成奥运冠军——这太震撼了
2. 作者说"你不会达到目标的高度，只会跌落到系统的水平"——这句话
   让我重新理解了系统的重要性
```

---

### 1 — 仍然存在的一个疑问

**目的：** 识别知识盲区，引导后续学习

**操作：**
- 写下1个你仍然不理解或想进一步探索的问题
- 这个问题将成为你下一步学习的方向

**示例：**
```
1个仍然存在的疑问：
- 书中说"身份认同比目标更重要"，但如果我的身份认同本身就
  是消极的（比如"我就是个不自律的人"），那该如何改变？
```

## 变体

### 变体1：对比型（3-2-1 Compare）

| 数量 | 内容 |
|------|------|
| **3** | 3个相似点 |
| **2** | 2个不同点 |
| **1** | 1个问题 |

> 适用于对比两篇文章、两个观点、两种方法的场景。

---

### 变体2：金字塔型（3-2-1 Pyramid）

| 数量 | 内容 |
|------|------|
| **3** | 3个学到的知识点 |
| **2** | 2个与已有知识的连接 |
| **1** | 1个与个人生活的关联 |

> 适用于建立知识间联系的场景。

---

### 变体3：阅读反思型

| 数量 | 内容 |
|------|------|
| **3** | 3个核心观点 |
| **2** | 2个我能应用的点 |
| **1** | 1个我想深入研究的问题 |

> 适用于读完一本书/一篇文章后的快速总结。

---

### 变体4：课堂反思型

| 数量 | 内容 |
|------|------|
| **3** | 3个关键概念 |
| **2** | 2个课堂上的例子 |
| **1** | 1个我还困惑的问题 |

> 适用于课堂教学的最后5分钟。

---

### 变体5：培训反思型

| 数量 | 内容 |
|------|------|
| **3** | 3个收获 |
| **2** | 2个明天就会尝试的行动 |
| **1** | 1个给培训师的反馈 |

> 适用于培训结束后的即时反思。

## 应用场景

| 场景 | 建议变体 | 时间 |
|------|----------|------|
| **课堂教学结尾** | 课堂反思型 | 最后5分钟 |
| **读书总结** | 阅读反思型 | 读完后5分钟 |
| **培训结束** | 培训反思型 | 最后10分钟 |
| **观看纪录片/TED** | 基本型 | 结束后3分钟 |
| **每周个人反思** | 基本型 | 10分钟 |
| **对比分析** | 对比型 | 灵活 |

## 3-2-1策略 与其他方法的对比

| 方法 | 复杂度 | 适用场景 | 特点 |
|------|--------|----------|------|
| **3-2-1** | ⭐ | 快速反思 | 数字约束，简洁有力 |
| **KPT** | ⭐⭐ | 周/月总结 | 行动导向 |
| **STAR** | ⭐⭐⭐ | 事件描述 | 情境还原 |
| **GRAI** | ⭐⭐⭐ | 项目复盘 | 系统分析 |
| **Gibbs** | ⭐⭐⭐⭐ | 深度反思 | 含情感维度 |

## 3-2-1策略的价值

1. **数字约束聚焦思考**：不是"随便写写"，而是被迫筛选最核心的3-2-1
2. **兼顾学习与反思**：不只记录学到了什么，还关注兴趣和疑问
3. **5分钟完成**：门槛极低，可以养成每日/每周的习惯
4. **疑问驱动学习**：那个"1个疑问"就是下次学习的起点

> **核心理念：简洁不是简陋，约束反而能逼出最精华的思考。**
