# SWBST法

## 概述

SWBST法是一种广泛应用于英语教学中的**叙事文本总结框架**，通过五个要素（Somebody / Wanted / But / So / Then）快速还原故事的核心线索。简单易学，适合中小学生及需要快速总结故事的场景。

---

## 五个要素

| 字母 | 英文 | 中文 | 核心问题 |
|------|------|------|----------|
| **S** | Somebody | 主角 | 故事的主角是谁？ |
| **W** | Wanted | 想要 | 主角想要什么？目标是什么？ |
| **B** | But | 但是 | 遇到了什么障碍/冲突？ |
| **S** | So | 所以 | 主角如何应对？做了什么？ |
| **T** | Then | 然后 | 结局如何？结果怎样？ |

---

## 详细说明

### S - Somebody（主角）

**目的：** 确定故事的核心人物

**关键问题：**
- 故事主要讲谁？
- 有哪些重要角色？

**示例：**
> "Country Mouse 和 City Mouse"

---

### W - Wanted（想要）

**目的：** 明确主角的动机和目标

**关键问题：**
- 主角想要实现什么？
- 主角的需求/渴望是什么？

**示例：**
> "Country Mouse 想体验城市的美食，City Mouse 想尝尝乡村的味道"

---

### B - But（但是）

**目的：** 识别冲突和障碍

**关键问题：**
- 什么阻止了主角实现目标？
- 出现了什么意外？

**示例：**
> "在城市里，他们不得不躲避猫和狗的追击，随时面临危险"

---

### S - So（所以）

**目的：** 描述主角的应对行动

**关键问题：**
- 主角做了什么来解决问题？
- 主角如何应对挑战？

**示例：**
> "两只老鼠四处躲藏，经历了一场惊心动魄的逃亡"

---

### T - Then（然后）

**目的：** 交代结局和结果

**关键问题：**
- 最后怎样了？
- 主角学到了什么？

**示例：**
> "Country Mouse 决定回到乡下，觉得平静简朴的生活比危险的奢华更珍贵"

---

## 完整示例

### 示例1：乡下老鼠和城里老鼠

> **Somebody:** Country Mouse and City Mouse **wanted** to have a meal together, **but** City Mouse thought the food wasn't fancy enough. **So** they went to the city but had to hide from a cat and dog. **Then** Country Mouse decided to go home and eat in peace.

### 示例2：龟兔赛跑

> **Somebody:** 兔子 **wanted** 证明自己跑得最快，**but** 在比赛中因为自信过头睡着了。**So** 乌龟一步一步坚持不懈地向前爬。**Then** 乌龟先到终点，赢得了比赛。

### 示例3：西游记（简化版）

> **Somebody:** 唐僧师徒四人 **wanted** 前往西天取经，**but** 一路上遇到了九九八十一难和各种妖魔鬼怪。**So** 孙悟空和师兄弟们各展神通，一路降妖除魔。**Then** 最终到达西天，取得真经，修成正果。

---

## 总结公式

将五个要素串联为一句话：

```
[Somebody] wanted [Wanted], but [But], so [So], then [Then].
```

> **某某**想要**某某**，但是**某某**，所以**某某**，然后**某某**。

---

## 应用场景

| 场景 | 说明 |
|------|------|
| **阅读理解** | 读完故事后快速总结情节 |
| **写作辅助** | 写故事前用SWBST搭建框架 |
| **口语表达** | 用SWBST复述故事 |
| **教学工具** | 帮助学生理解叙事结构 |
| **电影/剧集总结** | 快速概括剧情 |

---

## SWBST 的优势

1. **结构简单**：只有五个要素，零门槛
2. **逻辑清晰**：W→B→S 构成完整的冲突-解决链
3. **通用性强**：几乎所有叙事文本都可以套用
4. **培养结构思维**：帮助理解故事的基本结构

---

## 局限性

- ⚠️ 仅适用于**叙事类文本**（故事、小说、电影等）
- ⚠️ 不适合非虚构类文本（论说文、科普文章等）——这类文本建议使用 SAAC法 或 5W1H法
- ⚠️ 对于多线叙事的复杂故事，可能需要多次使用SWBST（每条线索一次）

---

## 进阶用法

### SWBST + 感受

在 Then 后面加上"我学到了什么"或"我的感受"：

```
Somebody wanted ___, but ___, so ___, then ___.
我学到的是：___
```

### SWBST + 多角色

对于多角色故事，分别填写不同角色的SWBST：

| 角色 | Wanted | But | So | Then |
|------|--------|-----|-----|------|
| 主角A | | | | |
| 主角B | | | | |
