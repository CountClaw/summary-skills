# 渐进式总结法（Progressive Summarization）

## 概述

渐进式总结法由知识管理专家 **Tiago Forte** 提出，是一种**分层递进**的总结方法。核心理念是：总结不是一次完成的，而是随着时间推移、理解加深，逐步压缩和提炼。每一层只对上一层进行约10%-20%的再精炼。

> 📖 来源：[fortelabs.com - Progressive Summarization](https://fortelabs.com/blog/progressive-summarization-a-practical-technique-for-designing-discoverable-notes/)

---

## 核心理念

> **"不要试图一次性写出完美的总结，而是让你的笔记随着时间'成长'。"**

- 每次接触笔记时，只需投入少量时间做进一步提炼
- 层层递进，从原始素材到一句话金句
- 最终目的是让笔记**高度可被发现、快速可调用**

---

## 五层结构

| 层级 | 名称 | 操作 | 时间投入 |
|------|------|------|----------|
| Layer 1 | 📝 原始笔记 | 记录原文/素材 | 记录时完成 |
| Layer 2 | 𝗕𝗼𝗹𝗱 加粗关键段落 | 对第一层加粗最关键的段落 | 10秒 |
| Layer 3 | 𝗛𝗶𝗴𝗵𝗹𝗶𝗴𝗵𝘁 高亮核心句 | 对第二层中加粗的部分高亮最精华的句子 | 30秒 |
| Layer 4 | 📋 Executive Summary | 用自己的话写一段超短总结 | 2-5分钟 |
| Layer 5 | 💎 一句话金句 | 压缩为一句话，随时可调用 | 随时 |

---

## 逐层详解

### Layer 1：📝 原始笔记

**操作：** 忠实记录原文或素材

**要点：**
- 不做任何加工，保持原始状态
- 可以是摘抄、截图、录音转文字
- 这是所有后续工作的"原材料"

**示例：**
```
冥想被越来越多的科学研究证实对心理健康有益。每天仅需10分钟
的冥想练习，就能显著降低焦虑和压力水平。哈佛大学的一项研究
发现，8周的正念冥想训练可以改变大脑结构，增强与注意力和情绪
调节相关的脑区活动。此外，冥想还有助于改善睡眠质量和提升免
疫力。建议初学者从每天5分钟开始，逐渐增加时间。
```

---

### Layer 2：𝗕𝗼𝗹𝗱 加粗关键段落

**操作：** 对Layer 1中最重要的段落加粗

**要点：**
- 只加粗约10-20%的内容
- 问自己："如果只看加粗的部分，能理解核心意思吗？"
- 时间投入：约10秒

**示例：**
```
冥想被越来越多的科学研究证实对心理健康有益。**每天仅需10分钟
的冥想练习，就能显著降低焦虑和压力水平。**哈佛大学的一项研究
发现，**8周的正念冥想训练可以改变大脑结构，增强与注意力和情绪
调节相关的脑区活动。**此外，冥想还有助于改善睡眠质量和提升免
疫力。建议初学者从每天5分钟开始，逐渐增加时间。
```

---

### Layer 3：𝗛𝗶𝗴𝗵𝗹𝗶𝗴𝗵𝘁 高亮核心句

**操作：** 在加粗的内容中，进一步高亮最核心的句子

**要点：**
- 只高亮加粗内容中的约10-20%
- 这是"精华中的精华"
- 时间投入：约30秒

**示例：**
```
冥想被越来越多的科学研究证实对心理健康有益。==每天仅需10分钟
的冥想练习，就能显著降低焦虑和压力水平。==哈佛大学的一项研究
发现，==8周的正念冥想训练可以改变大脑结构==，增强与注意力和情绪
调节相关的脑区活动。此外，冥想还有助于改善睡眠质量和提升免
疫力。建议初学者从每天5分钟开始，逐渐增加时间。
```

---

### Layer 4：📋 Executive Summary

**操作：** 用自己的话写一段超短总结（3-5句话）

**要点：**
- 必须**用自己的话**，不抄原文
- 包含核心观点 + 关键数据
- 时间投入：2-5分钟

**示例：**
```
📝 我的总结：
冥想只需每天10分钟就能显著降低压力，8周训练甚至能改变大脑
结构。建议从每天5分钟开始。核心价值：低成本、高回报的心理
健康工具。
```

---

### Layer 5：💎 一句话金句

**操作：** 压缩为一句话，方便随时调用

**要点：**
- 可以是原创提炼，也可以是原文中最精辟的句子
- 未来看到这句话就能回忆起完整内容
- 时间投入：随时

**示例：**
```
💎 "每天10分钟冥想，8周重塑大脑。"
```

## 层级递进图

```
Layer 1: 📝📝📝📝📝📝📝📝📝📝📝📝📝📝📝📝  (100% 原始)
                 ↓ 提炼10-20%
Layer 2:     𝗕𝗼𝗹𝗱 𝗕𝗼𝗹𝗱 𝗕𝗼𝗹𝗱 𝗕𝗼𝗹𝗱          (15-20%)
                     ↓ 再提炼10-20%
Layer 3:       𝗛𝗶𝗴𝗵𝗹𝗶𝗴𝗵𝘁 𝗛𝗶𝗴𝗵𝗹𝗶𝗴𝗵𝘁          (3-5%)
                         ↓ 自己的话
Layer 4:           📋 超短总结                (1段话)
                         ↓ 极致压缩
Layer 5:             💎 一句话                (1句话)
```

## 设计原则

### 原则1：不需要每次都做到第5层

```
大部分笔记 → 停在 Layer 2 就够了
重要的笔记 → 做到 Layer 3
极重要的笔记 → 做到 Layer 4-5
```

### 原则2：每层只需极短时间

| 层级 | 时间 |
|------|------|
| Layer 2 加粗 | 10秒 |
| Layer 3 高亮 | 30秒 |
| Layer 4 总结 | 2-5分钟 |
| Layer 5 金句 | 随时 |

### 原则3：让笔记"可被发现"

- 未来搜索笔记时，扫一眼加粗和高亮就能判断是否相关
- 不需要重读整篇笔记
- 这是渐进式总结的终极价值

## 应用场景

| 场景 | 说明 |
|------|------|
| **读书笔记** | 摘抄→加粗→高亮→总结 |
| **网页剪藏** | 保存→标记→总结 |
| **会议纪要** | 记录→加粗关键决策→总结行动项 |
| **学习笔记** | 课堂笔记→课后加粗→复习时总结 |
| **知识库管理** | 大量笔记分层管理，快速检索 |

---

## 常见错误

| ❌ 错误 | ✅ 正确做法 |
|---------|------------|
| 一次就想做到Layer 5 | 分层进行，允许笔记"慢慢成长" |
| 加粗了50%的内容 | 只加粗10-20%，否则失去意义 |
| Layer 4 还是抄原文 | 必须用自己的话重新表达 |
| 所有笔记都做到Layer 5 | 只对最重要的笔记做深度提炼 |
| 只做不回顾 | 定期浏览笔记，自然触发进一步提炼 |

---

## 渐进式总结的价值

1. **降低总结的门槛**：不需要一次性投入大量时间
2. **符合认知规律**：理解是逐步加深的，总结也应该逐步深化
3. **提高知识检索效率**：分层结构让快速浏览成为可能
4. **积累知识资产**：长期坚持，形成个人知识库的金矿

> **"最好的笔记不是写得最全的，而是未来最容易被找到和使用的。" —— Tiago Forte**
